# Test Automation using POM (Page Object Model) Design Pattern

 - A test automation solution for a simple Bidding Process:
   * Register Bid page
   * Place a Bid 
   * Edit a Bid
   * List registered Bids
  
  
  
  
 - Dependencies
   * Java 8
   * Maven
   * SpringBoot
   * Junit
   * Selenium 
   * Gerkhin
   * Cucumber
   * Chromium (chrome driver)
